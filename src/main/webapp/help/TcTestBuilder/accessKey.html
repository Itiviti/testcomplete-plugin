Specifies the access key assigned to your SmartBear account. The key will be used for licensing TestComplete/TestExecute instances that will run the tests.

To get the access key, login to the SmartBear License Management system at manage.smartbear.com with your SmartBear account and then click User > Access Key at the top right of the page.

Add the access key to the Credentials storage in Jenkins. Use the secret text credential type to store the key value.
